汉堡王餐厅招贤纳士

一、餐厅管理人员

任职要求：

1.  30周岁以下，大专以上学历。

2.  绝对的自信心，上进心和责任心。

3.  能适应倒班工作。

4.  能执行简单的电脑操作。

薪资待遇：

1.       月薪1600元 — 4000元

2.       每周休息1天

3.       转正后即享受养老保险、医疗保险

晋升发展前景：

见习助理 → 助理 → 餐厅副理 → 餐厅经理 → 公司管理人员

 

二、餐厅接待员

任职要求：

1.        28周岁以下，性格开朗。

2.        善于与人交往。

薪资待遇：

1.       月薪1500元 — 1700元

2.       转正后满一年，即享受养老保险、医疗保险

3.       每周休息1天

4.       全面系统的岗前培训

晋升发展前景：

见习接待员 → 正式接待员 → 星级接待员 → 资深接待员

 

三、服务员

任职要求：

1.        16—35周岁。

2.        能吃苦，有责任心

3.        男女不限

4.        可兼职

薪资待遇：

1.       小时计薪制（7.2元、8元、8.5元/小时，法定假日工作双薪）

2.       转正后满一年，即享受养老保险、医疗保险

3.       每周休息1天

4.       有充足的业余时间

5.       系统的岗前培训

6.       良好的晋升空间

晋升发展前景：

员工（收银员/服务员）→  星级训练员 → 全星级训练员 → 管理组长

 

四、产区固定员工

任职要求：

1.        50周岁以下，身体健康

2.        吃苦耐劳

3.        有责任心

薪资待遇：

1.       月薪1100元/月

2.       每周休息一天

 

五、清洁人员（晚班）

任职要求：

1.        吃苦耐劳

2.        有责任心

薪资待遇：

1.       月薪700元/月

2.       工作时间每晚4个小时左右，其它面议

 

六、出纳员 / 库房管理人员

任职要求：

1.        30周岁以下

2.        有责任心，能适应倒班

3.  能执行简单的电脑操作

薪资待遇：

1.       小时计薪制（7.2元、8元、8.5元/小时，法定假日工作双薪）

2.       转正后满1年，即享受养老保险、医疗保险

3.       每周休息1天
